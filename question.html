What are the different ways to select an element in the DOM?

What is the difference between innerHTML, innerText, and textContent ?
innerHTML includes HTML tags, innerText returns only visible text, affected by styles. textContent returns all text, including hidden text, but not HTML tags.
What is event delegation in the DOM?
In the DOM, "event delegation" is a technique where we can attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element
What is event bubbling in the DOM?
In the DOM (Document Object Model), "event bubbling" refers to the process where an event triggered on a specific element travels up through its parent elements in the DOM hierarchy, allowing each parent element to potentially handle the event as it "bubbles up" until it reaches the top of the document, essentially enabling a single event listener on a parent element to capture events from its child elements.
How do you create, add, and remove elements using JavaScript?
1. Create: document.createElement(), document.createTextNode()
2. Add: appendChild(), insertBefore()
3. Remove: removeChild(), element.remove()
