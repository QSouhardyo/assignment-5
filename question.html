What are the different ways to select an element in the DOM?
Using getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll
What is the difference between innerHTML, innerText, and textContent ?
innerHTML includes HTML tags, innerText returns only visible text, affected by styles. textContent returns all text, including hidden text, but not HTML tags.
What is event delegation in the DOM?
In the DOM, event delegation lets a parent element handle events from its child elements using a single event listener, avoiding multiple listeners on individual children.
What is event bubbling in the DOM?
In the DOM, event bubbling means an event triggered on an element propagates upward through its parents, allowing them to handle it.
How do you create, add, and remove elements using JavaScript?
1. Create: document.createElement(), document.createTextNode()
2. Add: appendChild(), insertBefore()
3. Remove: removeChild(), element.remove()
